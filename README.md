# DevOps Intern Task Solution

## О себе и мотивация (Why DevOps?)

Меня зовут [Твоё Имя]. Я хочу стать DevOps-инженером, потому что мне нравится находить баланс между разработкой и инфраструктурой. Мне интересно автоматизировать рутинные процессы, обеспечивать надежность систем и ускорять доставку кода до пользователей.

В этой задаче я продемонстрировал навыки работы с:
- Linux (Ubuntu)
- Веб-сервером Nginx
- Сетевой безопасностью (Firewall, SSL)
- Автоматизацией (Cron, Bash scripts)
- Контейнеризацией (Docker)
