# О себе и мотивация

Меня зовут Вадим. Я хочу стать DevOps-инженером, потому что мне нравится разрабатывать и так же поддерживать рабоспособность инфраструктуры.

В этой задаче я продемонстрировал навыки работы с:
- Linux (Ubuntu)
- Веб-сервером Nginx
- Сетевой безопасностью (Firewall, SSL)
- Автоматизацией (Cron, Bash scripts)
- Контейнеризацией (Docker)
